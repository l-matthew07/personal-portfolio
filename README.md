# Personal Portfolio Website

A modern, animated personal portfolio website built with React, Vite, and Framer Motion.

## Features

- 🎨 Beautiful animated transitions between pages
- 🖼️ Dynamic background images from Skelly collection
- 📱 Fully responsive design
- ⚡ Fast and optimized with Vite
- 🎭 Smooth animations powered by Framer Motion

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy images from skelly folders (if not already done):
```bash
node copy-images.js
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Project Structure

- `src/pages/` - Page components (Home, Projects, About, Contact)
- `src/components/` - Reusable components (Navigation)
- `src/styles/` - CSS stylesheets
- `public/images/skelly/` - Background images

## Pages

- **Home**: Welcome page with hero section
- **Projects**: Showcase of 4 projects with descriptions and links
- **About**: Personal information and skills
- **Contact**: Contact form and social links
